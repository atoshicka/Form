# Form

**Форма логина/регистрации пользователя с просмотром информации об его аккаунте, которая хранит данные на сервере с шифрованием пароля.**


## Стек

**Frontend**

• React 19 + TypeScript - ядро

• Vite - сборщик

• React Router DOM - навигация

• CSS - стилизация

• Lucide React - иконки

• React Hot Toast - уведомления



**Backend**

• Node.js - среда выполнения сервера

• Express - фреймворк для API роутов

• express-rate-limit - ограничение попыток входа

• bcrypt - шифрование паролей

• JSON Web Tokens (JWT) - токены авторизации

• pg (PostgreSQL) - драйвер для подключения к PostgreSQL

• dotenv - хранение секретных переменных

• cors - запросы между разными доменами



**Database**

• PostgreSQL (Neon) - база данных



**Deploy**

• Netlify (frontend)

• Render (backend)
