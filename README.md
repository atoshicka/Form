# Form

**Форма входа/регистрации пользователя с хэшированием пароля и JWT-токенами для аутентификации.**


*Прошу учесть, что сервер засыпает, если долго не взаимодействовать с этим приложением, поэтому убедительная просьба подождать подольше, пока сервер обработает ваш запрос. После первого раза все будет работать намного быстрее!*


## Стек

**Frontend**

• React 19 + TypeScript - ядро

• Vite - сборщик

• React Router DOM - навигация

• CSS - стилизация

• Lucide React - иконки

• React Hot Toast - уведомления



**Backend**

• Node.js - среда выполнения сервера

• Express - фреймворк для API роутов

• express-rate-limit - ограничение попыток входа

• bcrypt - шифрование паролей

• JSON Web Tokens (JWT) - токены авторизации

• pg (PostgreSQL) - драйвер для подключения к PostgreSQL

• dotenv - хранение секретных переменных

• cors - запросы между разными доменами



**Database**

• PostgreSQL (Neon) - база данных



**Deploy**

• Netlify (frontend)

• Render (backend)
